Hector by Autel

Written in under 2 weeks by a team of four

GitHub

Hector is designed to simplify agricultural surveying using drones produced by Autel Robotics. Given a polygon drawn by the user, the application will produce and store a survey route that will cover the entire selected area.

With the click of a button, Hector will convert this route into coordinates, transmit them to an active drone, and have it carry out the flight plan.

Severin's primary contributions to the project were:

  • Writing all SQLite DB interaction;
  • Writing all tests;
  • Setting up all data models;
  • Creating RecyclerViews and their adapters;
  • Paired programming on controller with Matt Duffin;
  • Contributing to user experience design;

FindMyPet!

Written in 24 hours by three Android developers and 3 UX designers

GitHub

This was designed and written in 24 hours as the winning entry for a Socrata hosted hackathon, where the goal was to use a City of Seattle API in a creative way. This app connects users to information about pets found by the city. Information is then viewed in a searchable format, including an image and contact information for the relevent animal shelter.

Information is saved to a local database, and additional API calls are only made if data is found to be over 12 hours old. Information will not be dropped until valid updates have been received.

Severin's primary contributions to the project were:

  • Writing all SQLite DB interaction;
  • Designing our system for requesting data;
  • Writing our API calls;
  • Writing a search function that simultaneously narrows data by: name; breed; color; region; gender;
